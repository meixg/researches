ECharts Advanced Y-Axis Compression Techniques

1. Dynamic Label Filtering Based on Value Range
技巧: 只显示特定范围内的值(如 0-100 和 500-600),隐藏中间区域
yAxis: { axisLabel: { formatter: function(value) { // Only show labels in range [0,100] or [500,600] if ((value >= 0 && value <= 100) || (value >= 500 && value <= 600)) { return value; } return ''; } } }
2. Break Axis Effect (Using Two Series)
技巧: 使用双 Y 轴创建断裂轴效果,中间部分被压缩
3. Custom Tick Marks with Visual Compression
技巧: 自定义刻度标记,在视觉上压缩中间部分
