ECharts Y-Axis Compression Demo

1. Standard Bar Chart (Original)
方法: 标准柱状图,显示所有 Y 轴刻度值
2. Hide Middle Labels (Show Only Min & Max)
方法: 使用 axisLabel.formatter 函数,只显示最小值和最大值,隐藏中间的标签
3. Show Every Nth Label
方法: 使用 formatter 函数,每隔 N 个刻度显示一个标签,减少标签密度
4. Custom Interval with Axis Split
方法: 使用 interval 和 splitNumber 控制刻度间隔和分割数量
5. Logarithmic Scale with Hidden Middle Labels
方法: 使用对数坐标轴(log)配合 formatter,处理大范围数据
