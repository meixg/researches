useFrontendTool

Hook

Registers a client-side tool with Zod-based parameter validation. Can include a custom render component to show progress in chat.

Usage Example

useFrontendTool({
  name: "sendEmail",
  description: "Send an email to a recipient",
  parameters: z.object({
    to: z.string().email(),
    body: z.string(),
  }),
  handler: async ({ to, body }) => {
    return `Email sent to ${to}`;
  },
  render: ({ args, status }) => (
    

       {status === 'Executing' ? 'Sending...' : 'Sent!'}
    

  )
});

useComponent

Hook

A convenience wrapper around useFrontendTool. It automatically generates a tool description and renders a React component using the tool's parameters as props.

Usage Example

useComponent({
  name: "WeatherCard",
  parameters: z.object({
    city: z.string(),
    temp: z.number()
  }),
  render: ({ city, temp }) => (
    

      {city}: {temp}°C
    

  )
});
